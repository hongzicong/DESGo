

## 证书结构

 - Certificate
    - Version
    - Serial Number
    - Algorithm ID
    - Issuer (CA’s name)
    - Validity
        - Not Before
        - Not After
    - Subject
    - Subject Public Key Info
        - Public Key Algorithm
        - Subject Public Key
    - Issuer Unique Identifier (Optional)
    - Subject Unique Identifier (Optional)
    - Extensions (Optional)
    - Certificate Signature Algorithm
    - Certificate Signature
